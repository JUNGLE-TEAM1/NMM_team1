# Harness Fixtures

Harness regression fixtures are generated by `scripts/test-harness.sh` in a temporary Git repository.

The generated cases cover:

- valid complete workspace
- unresolved Source of Truth proposal
- deferred Source of Truth proposal
- PR link without pushed branch
- missing pre-merge sync
- status workflow Source of Truth output
- local-only PR sync check
- missing harness regression script
- recorded harness test skip reason

The script uses temporary files so fixture tests do not mutate real workspace records or contact GitHub.
