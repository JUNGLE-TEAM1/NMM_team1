# Cross-platform smoke audit 계획

## 브랜치

- Branch: `docs/cross-platform-smoke-audit`
- Workspace: `docs/workflows/docs/cross-platform-smoke-audit`
- Created: 2026-06-24

## 목표

- macOS/Windows 로컬 개발 지원 기준 중 현재 실행 가능한 macOS Docker Compose 경로를 실제 smoke evidence로 검증한다.
- Windows WSL2/native Windows 검증은 현재 환경에서 실행 불가로 기록하고 후속 사람 조치를 남긴다.

## 범위

- macOS 현재 환경의 OS/shell/tool readiness 확인.
- Docker daemon/Compose readiness 확인.
- `scripts/smoke-container-app.sh` 실행.
- 하네스 검증 실행.
- audit report 작성.

## 범위 제외

- Windows WSL2 machine에서 직접 smoke 실행.
- native Windows PowerShell/CMD 지원 구현.
- PowerShell wrapper 또는 Python helper 추가.
- 제품 기능, API, schema, runtime behavior 변경.
