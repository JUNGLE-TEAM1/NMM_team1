# M3 metadata store plan 계획

## 브랜치

- Branch: `docs/m3-metadata-store-plan`
- Workspace: `docs/workflows/docs/m3-metadata-store-plan`
- Created: 2026-06-22

## 목표

- M3 시작 전에 첫 source type, metadata store, catalog API 범위를 결정하고 Source of Truth에 반영한다.

## 범위

- M3 첫 source type을 CSV/local file로 확정한다.
- M3 metadata store를 SQLite로 시작하되 `MetadataStore` 경계로 PostgreSQL/MongoDB 교체 가능성을 남긴다.
- M3 catalog API 최소 범위를 source create/list/detail, catalog list/detail, schema/row count/sample로 구체화한다.
- M3에서 제외할 file upload UI, PostgreSQL connection, pipeline execution 범위를 기록한다.

## 범위 제외

- 실제 backend/frontend 기능 구현.
- SQLite schema, migration, API handler 코드 작성.
- PR merge 이후 M3 구현 branch 생성.
