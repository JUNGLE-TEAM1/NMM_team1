Logical L11

Gold Deterministic Spec Compiler

Gold decision이 approved일 때만 Gold preview-only generation spec을 만든다. 승인되지 않은 Gold는 draft로 보존하지만 실행 operation은 비워둔다.

code: l6_compiler.pyphysical: l6spec: Gold optional

이 단계가 하는 일

  • approved Gold model만 aggregate operation으로 변환한다.
  • params는 input_ref, group_by, dimensions, measures, time_window, cardinality_guard 구조를 따른다.
  • not_requested/deferred/rejected 상태는 빈 operation으로 둔다.

코드와 산출물

코드tools/m3_contract/l6_compiler.py
주요 산출물gold_generation_spec.json
다음 입력M2 Gold preview execution, L14 Gold readiness

판단 기준

Gold는 사용자가 만들 수 있게 해야 한다. 무조건 생성하거나 무조건 추천을 실행 spec으로 확정하면 안 된다.

함수/코드 블록별 상세 해설
