Logical L6

Silver Recommendation Draft

Bronze를 Silver로 정제하기 위한 field별 action draft를 만든다. 이 단계의 결과는 실행 spec이 아니라 사용자가 수정할 수 있는 추천이다.

code: l4_recommendation.pyphysical: l4draft: Silver

이 단계가 하는 일

  • field별 target name, target type, action을 추천한다.
  • PII는 mask/hash/hide/query forbidden 정책 후보를 붙인다.
  • timestamp, numeric, bool, json, null normalization action을 분리한다.

코드와 산출물

코드tools/m3_contract/l4_recommendation.py
주요 산출물silver_policy_recommendation_draft.json
다음 입력L9 approval state, L10 Silver spec compiler

판단 기준

Silver draft는 구조 안정화와 안전한 query/catalog exposure가 목적이다. Gold semantic 판단과 섞지 않는다.
