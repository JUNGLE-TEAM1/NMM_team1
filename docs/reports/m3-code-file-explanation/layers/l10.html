Logical L10

Silver Deterministic Spec Compiler

승인된 Silver decision을 M2가 실행 가능한 preview-only transform spec으로 컴파일한다. M3가 Spark production write를 하지 않는다.

code: l6_compiler.pyphysical: l6spec: Silver

이 단계가 하는 일

  • Silver field action을 select/cast/rename/mask/hash/quarantine operation으로 변환한다.
  • preview_scope는 source_unit_ids, object_ids, stream_window_ids 중심으로 만든다.
  • write_mode를 preview_only로 고정한다.

코드와 산출물

코드tools/m3_contract/l6_compiler.py
주요 산출물silver_transform_spec.json
다음 입력M2 preview execution, L13 Silver evidence

판단 기준

production write, generated code execution, per-row AI call이 spec에 들어가면 안 된다. M3는 실행 엔진이 아니라 실행 가능한 계약 작성자다.
